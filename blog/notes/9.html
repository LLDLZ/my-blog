    nvm 安装及使用以及 node 环境配置

    # nvm 安装及使用以及 node 环境配置

    # 手动配置 nvm

    去官网下载 (opens new window) nvm-noinstall.zip

    之后解压一个你想要放置的地址，这个地址就是 nvm 的项目文件夹，比如我的就是D:\soft\nvm

    找到环境变量，配置环境

    • NVM_HOME nvm 地址目录
    • NVM_SYMLINK nvm 配置软链接的地址，该目录可以不存在。比如 D:\soft\nodejs

    追加内容到Path，追加的内容：

    %NVM_HOME%
%NVM_SYMLINK%

    # 将下载源换成国内的源

    到 nvm 目录的路径里看 是否有 settings.txt 文件，如果没有就新建

    root: D:\soft\nvm
path: D:\soft\nodejs
arch: 64
proxy: none

node_mirror: https://npm.taobao.org/mirrors/node/
npm_mirror: https://npm.taobao.org/mirrors/npm/

    到此 nvm 配置完成 在命令行 输入nvm --version nvm list 查看是否成功

    # nvm 常用指令

    • 查看已经安装的版本
      nvm list
    • 查看可安装的 node 版本
      nvm list available
    • 下载 对应的 node 版本
      nvm install 16.18.0
    • 激活 node 版本 nvm use 16.18.0

    # 多 node 版本，全局依赖安装多次问题

    准备一个存放 node 全局依赖的文件夹 nodeGlobal ，新建 node_cache node_global 文件夹

    # 修改 node 默认的安装位置

    • 查看 npm 全局模块的存放路径

      npm get prefix

    • 查看 npm 缓存默认存放路径

      npm get cache

    • 修改 npm 全局模块的存放路径

      npm config set prefix "D:\soft\nodeGlobal\node_global"

    • 修改 npm 缓存默认存放路径

      npm config set cache "D:\soft\nodeGlobal\node_cache"
