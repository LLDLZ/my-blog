```bash
    yarn
```

在项目根目录运行

### 目录内容
- `modules` 存放 `moduleCollection` 类和 `module` 类
- `plugin` 插件
- `helper.js` 实现 `mapState, mapGetters, mapActions, mapMutations` 方法的
- `index.js 、 index.cjs 、 index.mjs` 引用 Vuex 入口，导出了 `store 、install ...` 方法
- `mixin.js` `install` 核心，其实就是 `applyMixin` 方法
- `store.js` `Store` 实现核心。
